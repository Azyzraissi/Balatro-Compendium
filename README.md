Welcome to the **Balatro Compendium** for Raycast! This Extension is your ultimate local database for the game Balatro. With it, you can search through every piece of the Balatro collection in a <i>flush</i>. Whether you're hunting for Jokers, Tarot Cards, or any other collectible, Balatro Compendium has got you covered‚Äî**even offline**!

## üöÄ Features
- **Easy browsing**: Effortlessly explore the Balatro collection by categories.
- **Spoiler shield**: Decide if you want to see the unlocking conditions for certain elements.
- **Price display**: Check out the price of items available in the in-game shop.
